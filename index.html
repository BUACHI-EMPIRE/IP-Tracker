🌐 IP Address Fundamentals

📋 IP Address Types & Evolution

IPv4 (Internet Protocol version 4)

32-bit address format: 192.168.1.1

~4.3 billion unique addresses

Structure: 4 octets (8 bits each)
Example: 192.168.001.001
Limitation: Address exhaustion
IPv6 (Internet Protocol version 6)

128-bit address format: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

340 undecillion unique addresses

Structure: 8 groups of 16 bits
Benefits: Security, auto-configuration
Status: Growing adoption

🔗 Network Protocols & Architecture

🏗️ TCP/IP Protocol Stack (OSI Model)

Layer 7: Application Layer

HTTP, HTTPS, DNS, FTP, SMTP, SSH, Telnet, POP3, IMAP

Ports: 80, 443, 53, 21, 25
Data: Messages
Layer 4: Transport Layer

TCP (reliable), UDP (fast), SCTP, DCCP

Ports: 1-65535
Data: Segments/Datagrams
Layer 3: Network Layer

IP, ICMP, ARP, OSPF, BGP, IPv6

Addressing: IP
Data: Packets
Layer 2: Data Link Layer

Ethernet, WiFi (802.11), PPP, Frame Relay, ATM

Addressing: MAC
Data: Frames
Layer 1: Physical Layer

Copper, Fiber, Wireless, Cables, Connectors

Media: Physical
Data: Bits
📊 TCP vs UDP Comparison
Feature TCP (Transmission Control Protocol) UDP (User Datagram Protocol)
Connection Connection-oriented (3-way handshake) Connectionless
Reliability Guaranteed delivery, error checking No guarantee, no error checking
Ordering Guaranteed order No order guarantee
Speed Slower due to overhead Faster, minimal overhead
Use Cases Web browsing, email, file transfer Video streaming, gaming, DNS
Ports Well-known ports (0-1023) Dynamic ports (1024-65535)

🌐 Common Protocols Deep Dive

HTTP vs HTTPS
HTTP (Port 80):
• Plain text transmission
• No encryption
• Vulnerable to interception
HTTPS (Port 443):
• SSL/TLS encryption
• Secure data transmission
• Certificate verification
DNS (Domain Name System)
Record Types:
• A: IPv4 addresses
• AAAA: IPv6 addresses
• MX: Mail servers
• NS: Name servers
Process:
1. Check local cache
2. Query recursive DNS
3. Query authoritative DNS
Email Protocols
SMTP (Port 25/587):
• Sending emails
• Server-to-server
POP3 (Port 110/995):
• Download emails
• Remove from server
IMAP (Port 143/993):
• Sync emails
• Keep on server
File Transfer Protocols
FTP (Port 21):
• Basic file transfer
• No encryption
SFTP (Port 22):
• Secure file transfer
• SSH-based encryption
FTPS (Port 990):
• FTP over SSL/TLS
• Encrypted transfer

🌍 IP Geolocation & Privacy

IP addresses can reveal approximate geographic location, but with important limitations and privacy considerations.

Geolocation Accuracy
Country Level: 95-99% accurate
Region/State: 80-90% accurate
City Level: 60-80% accurate
Street Level: Rarely accurate
Privacy Limitations
• VPNs and proxies can mask real location
• Mobile devices may show carrier location
• Corporate networks may show HQ location
• Satellite internet may show ground station
Common Uses
• Content localization (language, currency)
• Fraud detection and security
• Compliance with regional laws
• Analytics and marketing

• Check WiFi password
• Move closer to router
• Change WiFi channel
• Update router firmware
